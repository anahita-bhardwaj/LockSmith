# LockSmith – Password Strength Analyzer 🔒

**LockSmith** is an interactive password strength analyzer that helps users create secure passwords. It features real-time strength visualization, entropy calculation, crack-time estimation, and password suggestions — all in a sleek, professional interface.  

Live demo: [LockSmith on GitHub Pages](https://anahita-bhardwaj.github.io/LockSmith/)

---

## **Features**

- **Generate secure passwords** instantly  
- **Real-time strength bar** with animated glow  
- **Entropy & crack time estimation**  
- **Password suggestions** for stronger security  
- Clean, modern design that works in any browser

## **How to Use**

1. Open LockSmith in your browser.  
2. Enter a password or click **Generate Secure Password**.  
3. Watch the strength bar and check suggestions.  
4. Use the password with confidence knowing it’s strong!
